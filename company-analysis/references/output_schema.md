# Micro Entity 输出格式

本文档定义"公司分析"skill 写入 F33-Micro 公司页面时的输出格式。

## 目标页面 Schema

遵循 `schema_entity_micro`（六段结构：故事 → 时间线 → 截面 → 三张表 → 盯什么 → 其它）。

"公司分析"skill 主要写入 **时间线** 和 **盯什么** 两段，必要时可更新 **截面** 段的关键运营数字。

## 时间线写入格式

时间线按日期正序排列（旧在上、新在下）。新增条目插入到正确的时间位置。

### K 线转折点条目格式

```markdown
- **YYYY-MM-DD [[公司名]] K线转折**：当日 <涨跌幅>，成交量 <相对描述>
  - **事件**：<公司发生了什么>
  - **逻辑**：市场交易的是 <逻辑描述>（<α 个股驱动 / β 行业共振>）
  - **横向**：同期 [[竞品名]] <走势描述>，<行业/个股判断>
  - [来源](url)
```

### 非 K 线基本面事件格式（标准时间线）

```markdown
- **YYYY-MM-DD**：<事件描述> [来源]
```

## 盯什么写入格式

覆盖更新。基于 K 线分析提炼 3-5 条即将验证的事件或数字：

```markdown
## 盯什么

- <即将验证的事件或数字 1>
- <即将验证的事件或数字 2>
- <即将验证的事件或数字 3>
```

每条的构成要素：
- **基于什么**：本次分析发现的模式或未完成的传导链
- **验证窗口**：大概什么时间能看到结果
- **证伪信号**：什么情况下判断失效

## 示例

```markdown
- **2026-03-15 [[大族激光]] K线转折**：当日 +8.2%，成交量放大 4 倍
  - **事件**：发布 12 英寸晶圆激光解键合设备通过中芯国际验证的消息
  - **逻辑**：市场交易的是半导体设备国产替代突破（α 个股驱动）——打破了日本 DISCO 垄断，打开了新增长曲线
  - **横向**：同期 [[华工科技]] 横盘 +0.3%，说明非行业共振，是大族独有的技术突破溢价
  - [公司公告](url)
```
